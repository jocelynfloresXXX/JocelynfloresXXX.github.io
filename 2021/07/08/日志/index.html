# 2021-07-03

1，第一天

如下

  • 1，乘车：
  • 买票，乘车，到北京。花了差不多四个小时，第一次来北京，在北京西站绕了几圈后才发现，把地铁 7 号线看成公交 7 路了，很无语。。。北京的生活就这么开始了.....
  • 2，租房：
  • 到北京西站后，联系到前几天在手机上问好的房东，但是，与自己之前想的差别很大，房子在昌平区，五环之外。光是去那就花了三个小时，拉着行李箱，终于到了那里，并且与房东说好准备租下时。给林哥打了个电话。林哥说离公司太远，每天跑来跑去就没有精力工作了。于是，拉着行李箱又做了三个多小时的地铁回去。心想着如果租不到房子就睡地铁站了，，，最后，还好朋友的亲戚正好在北京工作，在他的帮助下租到了房子，而且离公司比较近。想着等有时间了一定请他吃饭！：）
  • 3，房子
  • 在朋友的帮助下终于租到了离公司比较近的房子，到这时已经快 23：00 了，房子在城中村，和房东谈好价格拿到钥匙，价格比较便宜，并且不用交押金，觉得很好。当我打开房门时，被眼前的画面震惊了，房子很潮，里就一张床，一个凳子。没空调，没卫浴，最无语的还是公共卫生间！想了想还好吧，至少今晚不用睡地铁站或者天桥了：），于是，今天晚上去超市，买了个电风扇，和一些生活用品。洗完澡，躺在床上，就这样结束了算是北漂的第一天:》。

# 2021-07-04

1，第二天

如下

  • 1，星期天：
  • 来的第二天是星期天，林哥说 5 号才去公司报道，所以今天就没事，睡到早上 11 点多，就准备做地铁去公司以及深信服办事处那个公司看看。花了一上午的时间熟悉了公司地址后就准备回去，无意间在地铁上看到了有到圆明园的站，于是就多做了几站去了圆明园，门票 10 元，这天天很热，而且里面的人也很多，逛了很长时间，发现去西洋楼需要再买票才能进去，想了想时间也不早了，就赶紧回去了，准备以后走的时候再去看一下。第二天就这么结束了。。。

# 2021-07-05

1，第三天

如下

  • 1，入职：
  • 前一天晚上定了很多的闹钟，发现并没有什么用，还是没起那么早，，，淦！还好第一天去公司报道并没有迟到，这一天见到了林哥，林哥把公司合同拿过来让我看看并签一下。签完合同后就做到公司里了，过了一会，林哥就布置了学习任务，把链接发了过来，点开之后发现，里面其实就是介绍怎么工作的，参加项目的，和个人仪表的。并没有有关技术方面的知识。就这样一直看到了六点多钟，林哥说写完日志报告可以下班回去了，心想这就是 996？就这样做了一个小时的地铁回到了出租屋，实习的第一天就这么结束了：》

# 2021-07-06

1，第四天

如下

  • 1，实习：
  • 今天一大早就体会到公共卫生间的痛处，不过还好上班没有迟到，每天花一个小时左右上下班，今天还是像昨天一样，林哥布置了学习的任务，自己做在公司里看视频写笔记，很无聊的时候停下来听会歌，还行。到六点时把自己今天学习的内容以及知识点通过邮箱发给林哥后就下班回去了。。。。

# 2021-07-07

1，第五天

如下

  • 实习：
  • 今天又是看视频记笔记的一天，不过今天的学习内容终于有关于技术方面的知识了：)，今天写了很多笔记，也发现北京的人实在是很多，每天下班都要挤地铁，挤公交。而且交通也很复杂，下班回去后，在路上遇到了一大爷，就聊了起来，很难得，每天忙碌的日子里，终于可以和别人说说话了。在回去的路上林哥给了明天的任务，让去石景山区去取一下发票。搜了一下位置，有 20 多公里。

# 2021-07-08

1，第六天

如下

  • 实习：
  • 昨天林哥给了任务，于是今天早上并没有去公司，而是一大早起来，吃了早饭就骑车去了公交站，在公交车上看了一下行程，发现需要做 44 站，一共需要快两个小时，，，很无语。。。就这样做了 44 站，到了石景山区，问了好几个人最后才找到地方，拿到了发票之后，想到不能把发票弄折，就把它夹到了电脑里：)，于是又做了半个多小时的公交回到了公司，把发票给了郭姐，回到公司后，认识了刚刚来实习的河南安阳的老乡吧，到六点，提交了学习日报之后就回去了，今天是实习中最忙的一天，也算是实习的第一个 “项目” 吧。明天继续。
